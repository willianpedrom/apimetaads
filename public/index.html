Código de Integração Integrado

Insira o script abaixo no cabeçalho (<head>) de todas as páginas do seu site. Ele carregará o Pixel no navegador e enviará eventos para a Conversions API (CAPI) simultaneamente com deduplicação nativa.

Como disparar eventos com dados do cliente (Correspondência Avançada):
Nosso rastreador captura automaticamente os dados inseridos em qualquer formulário de cadastro, login ou lead. Mas você também pode disparar eventos customizados (Ex: Compra) diretamente no código JavaScript:

window.TrackSystem.track('Purchase', { email: 'cliente@exemplo.com', telefone: '5511999999999', nome: 'Willian Moreira', value: 149.90, currency: 'BRL', content_name: 'Curso Conversão CAPI' });
